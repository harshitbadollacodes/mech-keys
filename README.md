# Mech Keys ⌨

Bring the satisfying sound of a mechanical keyboard to your VS Code!

## Features

- 🔵 **Two sound profiles** — Cherry MX Blue and Lincoln Typewriter
- 🔊 **Volume control** — set your preferred volume level
- 🔄 **Click to switch** — click the status bar to cycle between profiles
- ⚡ **Low latency** — sounds preloaded into memory using Web Audio API

## Usage

1. Open the **MechKeys panel** from the activity bar (keyboard icon)
2. Click **"🎹 Click to Activate Sound"**
3. Start typing and enjoy the clicks!

## Commands

| Command                    | Description                        |
| -------------------------- | ---------------------------------- |
| `MechKeys: Toggle Sound`   | Turn sound on or off               |
| `MechKeys: Switch Profile` | Choose between Blue and Typewriter |
| `MechKeys: Set Volume`     | Set volume between 0 and 100       |

## Switching Profiles

Click the **MechKeys status bar item** at the bottom right to cycle between:

- 🔵 **Blue** — Cherry MX Blue, loud and clicky
- ⌨️ **Typewriter** — Lincoln Typewriter, retro thunk

## Settings

| Setting            | Default | Description              |
| ------------------ | ------- | ------------------------ |
| `mechkeys.enabled` | `true`  | Enable or disable sounds |
| `mechkeys.profile` | `blue`  | Sound profile            |
| `mechkeys.volume`  | `80`    | Volume level (0-100)     |
